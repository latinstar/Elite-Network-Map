# SigmaMap / an elite-network map

This repository contains data generated by an AI-powered scraper that maps networks of influence, power, and funding in contemporary politics.

Each folder represents a 'run' of the scraper characterized by specific generations of the script, containing Markdown files formatted for Obsidian.md. These files describe entities (people, organizations) and their relationships, with a focus on hidden connections and strategic influence.

## HOW TO CONTRIBUTE:
_Anyone_ can contribute to this repository. Simply follow the standard git contribution workflow by forking the repo, cloning it, making your changes, then commit them, push, and make a pull request. _All_ information that is relevant to the sope of the project, accurate, and non-redundant will be marged by the project's maintainers. 

Please include sources next to any changes you make so maintainers can easily verify the accuracy of information.
